About

Blended is a fork of Blender rebuilt around one stated identity: free 2D and 3D software tools, with an explicit focus on the craft of animation. Developed and published by CHJ 3 Productions LLC (Indiana).

Blended simplifies Blender’s data model (39 → ~19 ID types), introduces the .blended native project format, and keeps the full animation engine intact — depsgraph, keyframes, F-curves, timeline, armatures, NLA.

Blended is free and open-source software released under the GNU General Public License, version 2 or later (GPL-2.0-or-later). See the COPYING file included in this package for the full licence text. Source code is available at github.com/EvangAI-777/Blended.

Installation

Windows: Extract the downloaded .zip anywhere on your drive. Launch Blended by double-clicking Blended.exe. No installer required — the folder is self-contained and portable.

Linux: Extract the downloaded archive, then run the blended executable from the extracted folder.

macOS: Open the downloaded .dmg, drag Blended.app to your Applications folder, then launch it from there or the dock.

What’s Different

  • Simplified data model — fossils, UI-state types, and upstream deprecations removed
  • .blended native project format with platform integration
  • Import from .blend with a dropped-data manifest
  • Pre-5.0 rig compatibility for legacy Rigify rigs
  • Update notifications — background check at startup

Reporting Bugs
